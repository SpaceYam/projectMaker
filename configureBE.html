BE 생성하기

    laravel

                    After you have installed PHP and Composer, 
                you may create a new Laravel project via the Composer create-project command(CLI):
                composer create-project laravel/laravel example-app
                run by php artisan serve --port=8080

    Ruby on rails

                    download and install RubyGems at https://rubyinstaller.org/ then use CLI:
                gem install rails
                run by rails server

    ASP.NET

                    Open Visual Studio 2017 and select File menu -> New -> Project
                Create MVC Application, Create a New Project in Visual Studio, From the New Project dialog
                expand Visual C# node and select Web in the left pane, and then select 
                ASP.NET Web Application (.NET Framework)
                run by HomeController.cs > Index() > View() > create View

    JSP

                    install eclipse and Apache Tomcat then the following 
                File Menu -> New -> dynamic web project
                run by JSP project > Run as > Run on Server

    Spring Boot

                    create a new spring-starter-project with STS IDE or at https://start.spring.io/
                run by project right-click > Run As > Spring Boot App

    FastAPI

                    install Python 3.7+ then use CLI: pip install fastapi, pip install uvicorn
                run by uvicorn main:app --reload --host=0.0.0.0 --port=8000

    Express.js

                    install Node.js then use CLI: npm install express, npm init -y
                run by node serverfile.js
