개발한 소스 깃에 올리기(버전관리)

    1. 프로젝트에 .gitignore 파일을 생성하고 형상관리에서 제외할 파일/폴더 들을 등록합니다.
    ex) node_modules, build 등 빌드 파일, 불필요한 .log 파일 등

    2. 새 깃 Repo를 생성 후에 Repo의 https 주소를 복사합니다.



    3. 입력창에 깃주소를 입력하고, 프로젝트의 디렉토리에서 다음 CLI를 터미널에 실행합니다.   
    git init
git add -A .
git commit -m 'initial commit'
git remote add origin 
git push -u origin master
