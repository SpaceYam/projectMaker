GIT CLI의 활용

    1. 입력창에 작업한 브랜치를 입력하고, 다음 필요에 따라 다음 CLI들을 활용할 수 있습니다.   
    
- REPO 클론
git clone  
npm install
git pull origin develop --rebase   
git push origin develop         

- 스테이징 및 커밋
git status
git add -A 
또는 git add    
git commit -m  


- 브랜치 생성
git branch 

- (로컬 커밋 후) 원격 브랜치에 푸시
git push origin 

- origin 풀 받기
git checkout develop
git pull 

- 브랜치에서 origin 머지
git checkout 
git merge develop

- origin에서 브랜치 머지
git checkout develop
git merge 
git push (origin) 

- 브랜치 현황보기:    git branch -vva

- 로컬 브랜치 제거:   git branch -d 
- 원격 브랜치 제거:   git push origin --delete
    2. 깃 컨플릭 해결 
    - 상태확인:           git status
- 수정 후 스테이징:   git add .
  (특정 파일추가 시:  git add    )
  (특정 파일제거 시:  git rm  )
- 커밋:              git commit -m   
- 타임라인 확인:      git log --graph --decorate --oneline
